Verify OSPFv3

As shown in Figure 1, the show ipv6 protocols command is a quick way to verify vital OSPFv3 configuration information, including the OSPF process ID, the router ID, and the interfaces enabled for OSPFv3.

Use the Syntax Checker in Figure 2 to verify the OSPF protocol settings of R2 and R3 using the show ipv6 protocols command.

Use the show ipv6 ospf command to also examine the OSPFv3 process ID and router ID. This command displays the OSPF area information and the last time the SPF algorithm was calculated.
